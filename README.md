# telegram-bot-pooling-middle

Уровень 2. Средней сложности
Бот‑запись на услуги (парикмахер, стоматолог, консультации).
Функциональность:
Пошаговые диалоги (машина состояний, wizard).
Хранение данных в Postgres/MySQL.
Простая админка в самом боте для владельца (управление рассылками, просмотр статистики).
Интеграции:
CRM Bitrix24
Email, вебхуки по HTTP.
Технологии:
Go‑сервис + БД.
long polling.
Простая структура проекта (слои: transport → service → repository).
