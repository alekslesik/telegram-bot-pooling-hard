Уровень 3. Сложные / продвинутые боты
Примеры:
Платежные системы (оплата через Telegram Payments, подписки).
Маркетплейсы/магазины внутри бота.
Боты с личными кабинетами, аналитикой, статистикой.
Функциональность:
Много состояний, сложные воронки.
Платежи, баланс пользователей, реферальные программы.
Мультиязычность.
Инфраструктура:
Несколько сервисов (авторизация, биллинг, аналитика).
Redis для кеша, очереди (RabbitMQ, NATS, Kafka).
CI/CD, докеризация, мониторинг.
Заработок:
Серьезный чек, долгие проекты (недели–месяцы).
Долгосрочная поддержка/ретейнер.
